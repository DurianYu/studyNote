构建一个名为 "杜青林大冒险" 的向上跳跳乐的游戏网页。 游戏核心是 名为杜青林的卡通小人落在地面上时会往上弹起来。
核心玩法机制：
键盘A、D控制人物的左右移动，每往上跳入一层新地形就会生成一层更高且能够跳上去的地形，每次跳跃高度必定有能够踩上去的台阶
地形类型：湿地会减慢人物的移动速度、雪地会加快人物的移动速度（打滑）、长的短的、中间镂空的、火山地形则会使人物燃烧
道具系统：地形或者空中会生成道具，人物吃到道具时会播放一个音频文件作为触发的声音而且头顶会显示 杜青林大牛笔 的五彩斑斓闪闪发光的艺术字
道具类型：火箭-加速向上飞行一段时间（飞行期间也要生成台阶），五角星-免疫地形的影响一段时间，蘑菇-人物会变大一段时间
人物落地时会触发一些有趣的音效，例如放屁声、闪电、大笑等等
人物落地向上弹起时会产生金色波纹的特效
通关条件：往上跳过1000层地形就算成功，随着层数的增加地形难度越来越大
失败条件：人物落入当前视窗最底下没有地形的时候
游戏过程包含一首内置的欢快节奏电子合成器音乐（使用Web Audio API生成或Base64编码的短循环）
视觉风格与动效：
宫崎骏动画的美学风格：卡通人物、背景、地形、道具
背景类型：每200层换一个背景类型包括（蓝天白云、大气层、外太空、月球、太阳系）
界面采用 HUD 抬头显示风格，层数在右上角（稍微大一点醒目一点）
页面结构与交互：
开始界面：复古FC游戏风格 Logo，闪烁的 "INSERT COIN / START" 提示
结算界面：正中间显示 You are dead 的血红艺术字体，然后“重试”按钮放在下面
移动端适配：自适应、响应式设计
使用 requestAnimationFrame 确保 60FPS 流畅度
必须实现音画同步逻辑，设定延迟校准变量
所有图形通过 HTML5 Canvas 绘制
生成单一 HTML 文件，内嵌 CSS 和 JS
